# Helium language

This will be a repository for Helium, a Java-like programming language.

Helium has the primary extension `.he` (after Helium's elemental symbol, He)

I was greatly inspired to by this YouTube series on creating a compiler for a programming language called Hydrogen:
[Creating a Compiler](https://youtube.com/playlist?list=PLUDlas_Zy_qC7c5tCgTMYq2idyyT241qs&feature=shared)

This is the second iteration of this project, [see the first one here](https://github.com/Advait-Sen/Compiler)

Currently, the focus is to recreate the old programming language, and then to decide whether it will be compiled,
interpreted, or whatever the case may be.
